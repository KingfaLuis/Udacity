#### 属性信息：

1. 身份识别号
2. 诊断（M =恶性，B =良性）
3. 30 个特征

| -------- | ---------------------------- |
| 半径     | 从中心到边界点的距离的平均值 |
| 纹理     | 灰度值的标准偏差             |
| 周长     |                              |
| 面积     |                              |
| 平滑度   | 半径长度的局部变             |
| 紧凑度   | 周长2 / 面积 - 1.0           |
| 凹度     | 轮廓凹部的严重性             |
| 凹点     | 轮廓的凹部数                 |
