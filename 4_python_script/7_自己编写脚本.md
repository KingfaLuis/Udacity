# 自己编写脚本

在文本编辑器中打开全新的空文件，命名该文件并将其保存到放置 Python 课程文件的文件夹中。将以下代码放入该文件里。

```
how_many_snakes = 1
snake_string = """
Welcome to Python3!

             ____
            / . .\\
            \  ---<
             \  /
   __________/ /
-=:___________/

<3, Juno
"""


print(snake_string * how_many_snakes)
```

输出结果：

```

            / . .\
            \  ---<
             \  /
   __________/ /
-=:___________/

```

